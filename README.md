# EcoCircle

EcoCircle is a web platform designed to encourage individuals to take small, consistent eco-friendly actions and clearly demonstrate how those actions compound into meaningful environmental impact over time.

The platform focuses on awareness, participation, and motivation, proving that individual efforts—when tracked, rewarded, and visualized—can collectively make a real difference.

---

## What EcoCircle Does

EcoCircle enables users to:
- Sign in and manage a personal eco profile
- Join or create environmental initiatives
- Take eco challenges and earn GreenScore
- Share community insights and experiences
- Visualize the environmental impact created by their actions

---

## Core Features

### Authentication
- Simple sign-in system using browser local storage
- Maintains user sessions across all pages

### User Profile
Each user profile displays:
- User details
- Joined initiatives
- Challenges completed
- Community insights shared
- GreenScore progress

### Initiatives
Users can create or participate in initiatives such as:
- Tree plantation drives
- Clean-up campaigns
- Environmental awareness programs

Participation contributes to an increased GreenScore.

### Challenges
- Short-term eco challenges such as plastic-free weeks
- Completing challenges rewards higher GreenScore
- Duplicate participation is prevented

### Community Insights
- Users share real eco-friendly experiences
- Insights are visible to the entire community
- Encourages peer learning and motivation

### Impact Visualization
Graphs highlight the yearly impact of small actions, including:
- Trees planted
- Waste reduced
- CO₂ footprint reduction

All data is based on India-specific environmental averages for realistic representation.

### GreenScore System
- Starts at a base value
- Increases when users:
  - Join initiatives
  - Complete challenges
  - Share insights
- Acts as a progress tracker and motivation system
